# `starlight-automatic-overview-pages`

Insert new routes or append content to index.mdx pages in every folder so that users can get an overview.

## Documentation

Run the documentation locally by running the following command in your terminal:

```shell
pnpm run dev
```

Content can be found in the [`src/content/docs/`](./src/content/docs/) directory.
